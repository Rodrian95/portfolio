Trasy

Trasy is a logistic software that allows users to track their current logistic operation.
You can see a list of all the operations that the client registers and general information like the position in the map, the status, the provider of the operation and relevant dates
Trasy allows user to see with further details the current status of the operation that the user registers, in the detail page you can see the estimated dates of arrival(right) a map pointing the actual location of the operation(center) you can also upload invoices files, set the custom agent that is related to the operation, set the operations costs and see general information
Clients can register different related things to add to their operations like, providers, products and purchase orders
